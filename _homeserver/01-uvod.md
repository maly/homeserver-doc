Proč vlastní server místo krabiček a cloudů. Cíl: lokální služby, nízký footprint, plná kontrola.

### Požadavky
- Tichý HW (MSI Cubi 5 12M, i7, 32 GB RAM).
- Ubuntu Server LTS.
- Virtualizace přes KVM, HAOS ve VM.
- Kontejnery pro služby (Docker + Compose).
